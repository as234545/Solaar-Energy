# Solaar Energy - commercial website
Solaar Energy is a fictional company that sells and install solar panels to Companies, houses and farms. 


## Preview
[View Live Preview](http://solar.ksuit.org/ "Solaar Homepage") 


## Download and Installation 
 To begin using this template, choose one of the following options to get started:.<br/>
1- [Download the latest release on Start Bootstrap](https://getbootstrap.com/ "Bootstrap Homepage")  
2- Install via npm: `npm i Solaar-Energy` <br/> 
3- [Fork, Clone, or Download on GitHub](https://github.com/as234545/Solaar-Energy)   


## Usage

After downloading, simply edit the HTML and CSS files included with the template in your favorite text editor to make changes. These are the only files  you need to worry about, you can ignore everything else! To preview the changes you make to the code, you can open the `homebage.html` file in your web browser.
